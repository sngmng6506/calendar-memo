# Contributing

1. Read `AGENTS.md`.
2. Keep changes focused on the requested behavior.
3. Keep UI work aligned with the terminal dashboard style.
4. Run `npm run check` before handing off changes.
5. Update `README.md` or `docs/` when usage or architecture changes.

Do not add Python/Tkinter paths back into the app. AI and SNS integrations should be added later as isolated pages.
