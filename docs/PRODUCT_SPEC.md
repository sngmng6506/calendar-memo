# Product Specification

## Goal

바탕화면 달력의 날짜 셀에 남긴 체크리스트를 업무 원장으로 사용하고, 선택 기간의 기록을 보고서로 변환한다.

## Functional Requirements

### 월간 달력과 입력

- 월요일 시작, 7열, 고정 6주인 42개 날짜를 표시한다.
- 월 이동은 가로 중앙에 둔다.
- 날짜 셀은 카드와 흰 배경 없이 정렬과 여백으로 구분한다.
- `Enter`는 업무 저장 후 다음 빈 행에 포커스를 둔다.

### 미완료 복사

- 오늘 이전 모든 날짜의 미완료 업무를 오늘 마지막 순서로 복사한다.
- 원래 날짜의 업무는 삭제하거나 완료 처리하지 않는다.
- 같은 원본 업무는 같은 대상 날짜에 한 번만 복사한다.
- 복사본은 최초 원본 ID를 계보로 유지한다.
- 계보의 최신 사본이 완료되면 오래된 원본이 미완료여도 다시 복사하지 않는다.

### 주말과 대한민국 공휴일

- 토요일은 파란색, 일요일은 빨간색으로 표시한다.
- 대한민국 법정 공휴일과 대체공휴일은 빨간색으로 표시한다.
- `python-holidays`의 `KR` 달력을 우선 사용한다.
- 라이브러리가 없으면 2020~2050년 내장 데이터를 사용한다.

### Windows 바탕화면 통합

- WorkerW를 우선하고 Progman을 폴백으로 사용한다.
- 선택한 모니터 한 대의 좌표와 해상도만 사용한다.
- 일반 창에서 바탕화면 모드로 전환하면 현재 창이 있는 모니터를 자동 선택한다.
- 연결과 유지 시 layered alpha를 재적용한다.
- Explorer 재시작 후 재연결하고 실패하면 일반 창으로 복구한다.

### 로컬 저장과 보고서

- SQLite에 업무와 체크 상태를 저장한다.
- v0.4 DB에는 `origin_task_id`를 자동 추가한다.
- 일간·주간·월간 로컬 요약과 OpenAI 호환 LLM 보고서를 제공한다.
- API 키는 저장하지 않는다.

## Acceptance

1. 과거 여러 날짜에 미완료 업무를 만든다.
2. `미완료 복사`를 누르면 모두 오늘로 복사되고 원본은 남는다.
3. 버튼을 다시 눌러도 중복되지 않는다.
4. 일반 창을 보조 모니터로 옮긴 뒤 바탕화면 모드로 전환하면 같은 모니터에 고정된다.
5. 토요일은 파란색, 일요일과 대한민국 공휴일은 빨간색으로 보인다.
