# Roadmap

로드맵 항목은 승인된 기능이 아니라 후보 목록이다. 구현 전 `AGENTS.md`의 Scope Guard 변경 승인이 필요하다.

## v0.1 MVP

- 월간 달력
- 인라인 체크리스트
- SQLite 저장
- 어제 미완료 이동
- 일간·주간·월간 로컬 요약
- OpenAI 호환 LLM 보고서
- Windows/Linux CI

## v0.2 Candidate

- 날짜 셀 내부 스크롤 또는 접기
- 보고서 이력 조회
- CSV/Markdown 내보내기
- Windows 단일 실행 파일 배포 자동화
- 자동 업데이트 없이 수동 릴리스 패키지

## Explicitly Deferred

로그인, 클라우드 동기화, 협업, 태그·프로젝트, 반복 일정, 외부 캘린더 연동은 제품 정체성을 다시 검토하기 전까지 보류한다.
